# PKSIG

Sistema ERP operacional para assistência técnica, suporte e reparo.

---

# Objetivo

O projeto PKSIG nasce com o objetivo de estruturar operacionalmente o fluxo real de uma assistência técnica moderna.

A proposta do sistema é:

- reduzir burocracia;
- melhorar organização operacional;
- melhorar gestão da bancada;
- melhorar acompanhamento financeiro;
- melhorar rastreabilidade dos serviços;
- refletir a rotina real do técnico.

# Conceito Central

A principal decisão arquitetural atual do projeto é:

```text
+ Cliente é quem.
+ Equipamento é o quê.
+ Serviço é o que está acontecendo.
+ Status é onde o serviço está no fluxo.
+ Financeiro é o dinheiro movimentado.
```
